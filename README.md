
# Kunskapskontroll 2

Detta projekt är en del av Kunskapskontroll 2 i kursen Statistik och dataanalys. Syftet är att analysera bankkunders beteende och dela in kunder i tydliga kundgrupper för att skapa beslutsstöd för bankens marknadschef.

Projektet innehåller:
- Datatvätt och klustring i Python
- Visualisering och analys i Power BI
- Presentation av insikter och rekommendationer

Python-koden har exekverats och resultatet används vidare i Power BI.
